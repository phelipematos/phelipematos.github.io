# PhelipeDev – Portfólio

Portfólio pessoal simples e elegante, pronto para publicar no GitHub Pages.

## Estrutura

- `index.html`
- `css/style.css`
- `imagens/` (opcional)

## Como publicar no GitHub Pages

Siga exatamente estes passos:

1) Renomeie o repositório para `phelipematos.github.io`.
2) No GitHub, vá em **Settings -> Pages**.
3) Em **Build and deployment**, selecione **Deploy from a branch**.
4) Em **Branch**, selecione `main` e a pasta `/ (root)`.
5) Clique em **Save**.
6) Aguarde a publicação e confirme o link exibido (algo como `https://phelipematos.github.io`).

## Personalização rápida

- Substitua o texto dos projetos e do contato em `index.html`.
- Adicione sua foto na pasta `imagens/` e atualize o placeholder no HTML.
- Ajuste cores e tipografia em `css/style.css`.
